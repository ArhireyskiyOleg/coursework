## 🚀 Installation and usage

### ✅ API parts
#### Steps

1) Install mongodb
2) Install python3
3) Run in shell:

```bash
python -m ensurepip --default-pip
```

```bash
pip install -r requirements.txt
```

### ✅ App parts
#### Steps

1) Install nodejs
2) Install yarn
3) Run in shell:

```bash
yarn install
```

#### Build
To build the app for production run in shell:

```bash
yarn build
```

### ✅ Run
```bash
yarn start
```
