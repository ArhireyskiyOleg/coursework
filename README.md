# Coursework

- python
- mongodb
- express
- react

## API parts
### Deps

1) install mongodb
2) install python3
3) run in shell:

```bash
python -m ensurepip --default-pip
```

```bash
pip install -r requirements.txt
```

### Run
```bash
uvicorn app.main:app --reload
``` 

## Full-stack parts
### Deps

1) install nodejs w/o npm
2) install yarn
3) run in shell:

```bash
yarn install
```

### Build
```bash
yarn build
```

### Run
```bash
yarn start
```
